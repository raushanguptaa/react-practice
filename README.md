# React Portfolio Landing Page

A simple and responsive portfolio website built using React and Tailwind CSS.

## Features

- Responsive Navbar
- Hero Section
- About Section
- Skills Section
- Projects Section
- Contact Section
- Footer

## Tech Stack

- React
- Tailwind CSS
- Vite
